# blazebury
Blazebury is a music app written in NodeJS and Electron.

## features
- Absolutely no DRM
- More than one scraper (Deezer, Lastfm**)
- Can work anywhere
- In-app lyrics
- Discord Rich Presence support

** = requires API key
